About

Welcome to the world's first web based nuclear physics simulation software. As the name would suggest, the Geant4 based simulation software gemc is used on the back-end. gemc was developed by Maurizio Ungaro, Ph.D, at the Thomas Jefferson National Accelerator Facility in Newport News, Virginia. More information about gemc can be found on the official website: gemc website.

The official GitHub repository can be found here: gemcWeb GitHub
